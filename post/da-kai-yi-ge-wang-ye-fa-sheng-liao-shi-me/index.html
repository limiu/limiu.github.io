打开一个网页发生了什么

2020-11-11 1 min read # HTTP请求 # 前端

在浏览器上打开页面 https://studies.cs.helsinki.fi/exampleapp/notes 时会发生什么。

  • 浏览器使用 HTTP GET 请求 从服务器中获取定义内容和页面结构的HTML
  • HTML 代码中的 Links 标签 会让浏览器获取CSS样式表 main.css
  • 以及 JavaScript 代码文件 main.js
  • 浏览器执行 JS代码 ，并且向地址：https://studies.cs.helsinki.fi/exampleapp/data.json 发出HTTP GET请求，请求返回 data.json 数据
  • 浏览器获取数据后，执行一个event handler（发生了什么事件，执行什么函数） 事件处理程序，- 使用DOM-API将Note 渲染到页面
